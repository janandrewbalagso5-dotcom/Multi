# ASP.NET Web Forms School Project

## Requirements
- Visual Studio 2019 or later
- .NET Framework
- IIS Express (included with Visual Studio)

## How to Run
1. Clone the repository
2. Open the `.sln` file in Visual Studio
3. Press F5 to run the project
4. The site will open in a browser using IIS Express

## Notes
This project uses ASP.NET Web Forms and cannot be hosted on GitHub Pages or Vercel
because it requires server-side execution.
